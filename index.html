To-do List

What can you do with this list?

  • Add or remove elements using website UI
  • Edit element contents by double clicking the text
  • Drag elements around with mouse
  • Mark items as complete by clicking on the checkmark near an element
  • You can refresh the page, close your browser, or even restart your device - your list will stay intact, keeping the same layout and content

How I built this and what I used

This website is built with vanilla JS, HTML, and SCSS. I'm using local storage to save the layout user creates. Local storage is updated after every removal, addition, edition, or completion status change of any element. I'm using Google Fonts' Open Serif as the page font, Font Awesome for icons, colorhunt.co for the color palette.

Enjoy Using the site! I suggest clicking all the buttons and refreshing your page multiple times. It really works!

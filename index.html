To-do List

This is a fully responsive to-do list with advanced functionality and intuitive UI

What can you do with this list?

  • Add or remove elements using the interface
  • Edit elements by double clicking or double tapping the text
  • Element dragging by dragging with mouse or by holding a touch on an element and dragging it
  • Mark elements as complete or incomplete using the checkbox icon
  • You can close the website and reopen it later with having your items left intact. Completion status, text content, and the layout of all items is stored in browser local storage

Built with HTML SCSS and vanilla Javascript
